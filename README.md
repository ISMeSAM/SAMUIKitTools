# SAMUIKitTools

##使用方法
使用CocoaPods
```objc
pod 'SAMUIKitTools', '~> 0.0.5'
```
	#import "SAMUIKitTools.h"



##SAMUIKitTools结构

### 目录结构


```
SAMUIKitTools

	├── Categories				#分类部分

	├── UITools           		 #UI部分       

	├── Categories.h     		  #分类部分头文件      
     
	├── UITools.h    				#UI部分头文件 

	└── SAMKeys.h      			#宏工具

```
